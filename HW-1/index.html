Home work: Git

Base:

  • git config --global user.name "Your name" - глобально установить имя пользователя;

  • git config --global user.email "Your@email" - глобально установить email пользователя;

  • git init - инициализация репозитория на github;

  • git status - отображения состояния файлов;

  • git add - добавляет содержимое репозитория в определенное место, с которого потом будет сделан слепок проекта;

  • git commit - делает слепок проекта с того же места, куда и были добавлены файлы командой git add;

  • git log - выводит список коммитов в данном репозитории в обратном хронологическом порядке;

  • git checkout - перемещение указателя между коммитами;

  • git branch - управление ветками;

  • git push - выгрузка содержимого локального репозитория на удаленный;

  • git pull - выгрузка с удаленного репозитория на локальный;

  • git merge - объеденяет текущую ветку с такой же веткой из репозитория;

  • git clone - копировать репозиторий.

.gitignore:

Нужен для скрытия файлов и папок от системы контроля версий Git. Обычно скрывают конфигурационные файлы (особенно с паролями), временные файли и папки.


Advanced:

  • git revert - отменяет необходимый коммит;

  • git reset - отменяет индексацию выбранных файлов, то есть убирает их из той области с которой будет сделан слепок;

  • git rebase - объединяет изменения сделанные в одной ветке;

  • git cherry-pick - переносит определенные коммиты из одного места репозитория в другое;

  • git stash - откладывает изменения в робочей копии, чтобы применить их позже.
